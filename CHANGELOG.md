# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Browser extension for Chrome and Firefox
- Documentation site with Docusaurus
- Contributing guide and Code of Conduct
- Changelog

## [0.3.0] - 2026-06-16

### Added
- Internationalization support (11 languages)
- Mobile app foundation (React Native)
- Browser automation
- Terminal autocomplete
- Voice recognition and text-to-speech
- Image analysis for screenshots and diagrams
- Performance tests
- E2E tests with Playwright

### Changed
- Improved codebase indexing performance
- Enhanced documentation RAG
- Better terminal autocomplete suggestions

### Fixed
- Various bug fixes and improvements

## [0.2.0] - 2026-06-15

### Added
- VS Code extension
- JetBrains plugin
- MCP server
- Extension system
- Onboarding system
- Integrated debugging
- Monitoring with Prometheus
- Security improvements (JWT, environment variables)

### Changed
- Improved web UI with Monaco editor
- Enhanced CLI with 50+ commands
- Better documentation

### Fixed
- Various bug fixes and improvements

## [0.1.0] - 2026-06-14

### Added
- Initial release
- Backend API with FastAPI
- Authentication with JWT
- Database support (SQLite/PostgreSQL)
- WebSocket for real-time collaboration
- Desktop app with Tauri
- Web UI
- CLI
- TypeScript SDK
- Codebase indexing
- Documentation RAG
- Basic tests
