Anvil Features

Everything you need in a modern coding assistant — and more.

Unique Features (Only Anvil Has These)

Verify Loop UNIQUE

Every change is automatically verified. Other agents generate code and hope it works. Anvil generates, verifies, and fixes until it's correct.

Error Recovery UNIQUE

When verification fails, Anvil doesn't just report the error. It diagnoses the issue and automatically fixes it, then re-verifies.

Multi-Interface UNIQUE

Use Anvil your way: Desktop app, Web UI, CLI, VS Code extension, JetBrains plugin, or MCP server. All interfaces share the same powerful backend.

Extension System UNIQUE

Extend Anvil with custom skills, tools, and integrations. Build your own or install from the community.

Onboarding Tour UNIQUE

New to Anvil? Interactive guided tours teach you the basics in minutes. No other assistant has this.

Integrated Debugging UNIQUE

Breakpoints, step-through debugging, variable inspection, and error analysis all built-in. No other coding assistant has this.

Modern IDE Features

Inline Edit (Cmd/Ctrl+K)

Edit code directly in the editor. Select code, describe the change, and Anvil applies it.

Tab Autocomplete

Ghost text suggestions as you type. Press Tab to accept, Escape to dismiss.

Slash Commands

Quick actions in chat: /test, /fix, /explain, /refactor, /verify, /debug.

@-mentions

Reference files, symbols, and documentation in chat with @filename or @symbol.

Git Integration

Auto-commit, diff viewing, undo, and AI-generated commit messages.

Custom Rules

Define AI behavior with custom instructions. Create rules for your coding style.

API Cost Tracking

See token usage and costs in real-time. Monitor your API spending.

Checkpoints/Undo

Rollback AI changes with one click. Safety net for all AI edits.

Codebase Context

Index and search your entire codebase. Anvil understands your project structure.

Memory System

Anvil remembers your preferences across sessions. Persistent learning.

Real-time Collaboration

Work together with WebSocket support. See other users' cursors and changes.

Monitoring & Metrics

Prometheus metrics for observability. Monitor performance and usage.

Interfaces

Desktop App

Native Tauri application for macOS, Windows, and Linux. Auto-updates, standalone.

Web UI

Browser-based IDE with Monaco editor. Full-featured, responsive design.

CLI

Full command-line interface. 50+ commands for all operations.

VS Code Extension

Editor integration with context menu, sidebar, and keyboard shortcuts.

JetBrains Plugin

Works with IntelliJ, PyCharm, WebStorm, and all JetBrains IDEs.

MCP Server

Model Context Protocol server for AI agent integration.
